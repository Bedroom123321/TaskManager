| TC-TASK-04 | Отметка выполнения задачи свайпом | 1. На главном экране найти задачу в списке.<br>2. Свайпнуть задачу вправо. | Задача отмечается как выполненная. Визуально изменяется (зачеркивается или меняет цвет). | | |
| **Модуль "Календарь"** |
| TC-CAL-01 | Создание задачи из календаря | 1. В календарном представлении нажать на дату (25 ноября).<br>2. Нажать "+" для создания задачи.<br>3. Ввести название "Встреча".<br>4. Сохранить. | Задача создается с датой 25 ноября. Отображается в календаре на выбранной дате. | | |
| TC-CAL-02 | Создание повторяющейся задачи | 1. Создать задачу "Зарядка".<br>2. Включить "Повторять".<br>3. Выбрать "Ежедневно", время 07:00.<br>4. Сохранить. | Задача создается с повторением "Ежедневно". В календаре отображается на каждый день. | | |
| TC-CAL-03 | Перенос задачи drag-and-drop | 1. Открыть недельный вид календаря.<br>2. Найти задачу на понедельнике.<br>3. Перетащить задачу на среду. | Задача перемещается с понедельника на среду. Дата задачи автоматически обновляется. | | |
| **Модуль "Уведомления"** |
| TC-NOT-01 | Получение push-уведомления | 1. Создать задачу с датой через 2 минуты.<br>2. Включить напоминание "За 1 минуту".<br>3. Свернуть приложение.<br>4. Дождаться уведомления. | За 1 минуту до времени задачи приходит push-уведомление с названием задачи. | | |
| TC-NOT-02 | Переход к задаче из уведомления | 1. Получить push-уведомление.<br>2. Нажать на уведомление. | Приложение открывается и сразу отображает детали соответствующей задачи. | | |
| **Модуль "Категории и приоритеты"** |
| TC-ORG-01 | Создание категории с цветом | 1. Открыть настройки категорий.<br>2. Нажать "Добавить".<br>3. Ввести "Работа", выбрать синий цвет.<br>4. Сохранить. | Категория "Работа" создается с синим цветом. Доступна при создании задач. | | |
| TC-ORG-02 | Фильтрация по приоритету | 1. Создать задачи с разными приоритетами.<br>2. Открыть фильтры.<br>3. Выбрать "Высокий приоритет". | Отображаются только задачи с высоким приоритетом. Остальные скрыты. | | |
| **Модуль "Синхронизация"** |
| TC-SYNC-01 | Создание задачи в автономном режиме | 1. Включить режим самолета.<br>2. Создать задачу "Офлайн задача".<br>3. Сохранить. | Задача успешно создается локально. Отображается индикатор "Не синхронизировано". | | |
| TC-SYNC-02 | Автоматическая синхронизация | 1. Создать задачу офлайн.<br>2. Включить интернет.<br>3. Дождаться синхронизации. | Задача автоматически синхронизируется. Индикатор исчезает. | | |
| **Производительность** |
