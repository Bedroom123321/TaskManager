# 2 Варианты использования

<a name="create_task"/>

## 2.1 Создать задачу

**Описание.** Пользователь создает новую задачу, указывает её название, описание, срок, приоритет, категорию, опционально добавляет подзадачи и напоминание.
**Основной поток.**

1. Приложение отображает форму создания задачи.
2. Пользователь вводит параметры задачи.
3. Пользователь сохраняет задачу.
4. Задача появляется в общем списке задач.
<a name="edit_task"/>

## 2.2 Редактировать задачу

**Описание.** Редактирование параметров существующей задачи.
**Основной поток.**

1. Пользователь выбирает задачу.
2. Приложение отображает форму редактирования.
3. Пользователь вносит изменения и сохраняет.
<a name="delete_task"/>
